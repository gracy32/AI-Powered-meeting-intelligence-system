# 🤖 AI-Powered Meeting Intelligence System

An intelligent meeting analysis application built using **Python**, **Streamlit**, and **Google Gemini AI**. The system processes meeting transcripts and automatically generates meaningful insights such as summaries, action items, and key decisions, helping teams save time and improve productivity.

---

## 📌 Features

- 📝 Automatic Meeting Summary
- ✅ Extracts Action Items
- 🎯 Identifies Key Decisions
- 🤖 Powered by Google Gemini AI
- 💻 Interactive Streamlit Web Interface
- ⚡ Fast and User-Friendly

---

## 🚀 Installation

### Clone the repository

```bash
git clone https://github.com/gracy32/AI-Powered-meeting-intelligence-system.git
```

### Navigate to the project

```bash
cd AI-Powered-meeting-intelligence-system
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Create a `.env` file

```text
GOOGLE_API_KEY=your_google_gemini_api_key
```

### Run the application

```bash
streamlit run app.py
```

---

## 📷 How It Works

1. Launch the Streamlit application.
2. Paste the meeting transcript into the text area.
3. Click the Analyze button.
4. The AI generates:
   - Meeting Summary
   - Action Items
   - Key Decisions
